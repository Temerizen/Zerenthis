# Zerenthis GitHub Site

This is a premium static Zerenthis website package built to be easy to drop into a GitHub repo.

## Files
- `index.html` - the full website
- `assets/zerenthis-logo.png` - logo image
- `.nojekyll` - helps GitHub Pages serve files cleanly

## Easiest GitHub setup
1. Create a new GitHub repo
2. Upload all files from this zip to the repo root
3. In GitHub, open **Settings > Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save

GitHub Pages will publish your site.

## Notes
This is a polished static frontend shell.
It does not include a backend, auth, payments, or database.
